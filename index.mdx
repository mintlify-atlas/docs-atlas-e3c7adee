The Buildkite Agent is a small, reliable, and cross-platform build runner that lets you run automated builds on any device or network you control. It polls Buildkite for work, runs build jobs in a secure bootstrap environment, and streams logs and artifacts back to Buildkite in real time.

## What the agent does

The agent bridges your own infrastructure with the Buildkite platform. Once started, it:

1. Registers itself with Buildkite using your agent token
2. Polls for scheduled jobs (or receives them via streaming dispatch)
3. Runs each job inside a bootstrap shell, checking out code and executing your pipeline steps
4. Streams log output and uploads artifacts back to Buildkite
5. Reports the final job status and moves on to the next job
